About NIDPS (Network Intrusion Detection and Prevention System)

This system is designed to monitor live network traffic in real time, detect potential intrusions or anomalies using machine learning, and proactively block malicious IP addresses to protect the system.

Key Features

  • Real-time packet capture using PyShark
  • Machine learning-based classification (Decision Tree)
  • Automatic and manual IP blocking
  • Live dashboard visualization with Flask + Socket.IO
  • Blocked IP logging and Supabase integration for data storage

Technologies Used

  • Python (Flask, Scikit-learn, PyShark)
  • HTML, CSS, JavaScript (Frontend)
  • Flask-Socket.IO for real-time updates
  • Supabase for cloud-based database storage
  • Linux Firewall (iptables / UFW)

How It Works

The NIDPS continuously listens to network traffic on a selected interface. Each packet is analyzed and classified as either Normal or Anomalous using a pre-trained Decision Tree model. If a source IP sends too many anomaly packets, the system blocks it automatically. All blocked IPs and their metadata are stored in a remote Supabase database, and admins can manage them through this dashboard.

Intended Use

This system is intended for educational, research, and small-scale network security environments. It can be deployed on Linux-based virtual machines or servers to enhance security visibility and automate early-stage intrusion response.
